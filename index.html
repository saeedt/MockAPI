Visualroo

A web application to retrieve and visualize data using the Mockaroo.com API

How to set up your Mockaroo schema in Visualroo

Visualroo works with any registered user's Mockaroo account. In Mockaroo create the schema, format it in JavaScript Object Notation (JSON), and save the schema. Start Visualroo, copy the schema link and API key to Visualroo, click the test API button, choose x and y variables, and click the Retrieve Data button.
